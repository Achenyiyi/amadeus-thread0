## Recommended Writing Order

Do not write the thesis linearly. Use this order:

1. fix abstract and introduction wording
2. finish Chapter 3 system design
3. finish Chapter 4 experiment chapter with current automatic evidence
4. run user study and fill subjective results
5. then write related work and conclusion

## Rule

Every paragraph in the experiment chapter should map to one of:

- a report file
- a code path
- a qualitative case
- a user-study artifact

If it maps to nothing, rewrite or remove it.
