# 第二章 相关工作初稿

## 2.1 角色对话智能体研究

随着大语言模型在开放域对话中的广泛应用，面向特定人格或角色设定的 role-playing agent 逐渐成为研究热点。现有工作通常通过系统提示词、角色描述文本或示例对话来约束模型输出，使其在短轮交互中模拟目标角色的口吻与表达风格。这类方法实现成本较低，易于快速原型化，因此在学术研究和应用实践中均较为常见。

然而，仅依赖 prompt 的角色扮演系统往往存在明显局限。首先，模型在面对复杂任务、工具调用或外部知识检索时，容易从角色表达退化为通用助手腔。其次，角色扮演效果通常集中在单轮或短轮交互中，对长期关系演化、共同经历与记忆连续性的支持不足。因此，越来越多研究开始关注如何将状态、记忆和判别机制引入角色智能体系统，以降低人格漂移并增强长程互动体验。

与这类工作相比，本文并不将角色一致性问题视为“角色 prompt 写得好不好”的问题，而是视为一个有状态智能体的系统设计问题。本文通过双阶段生成、人格偏移判别、世界线记忆结构和正式评测闭环来控制角色表现，使角色一致性从文本描述层提升到后端结构层。

## 2.2 面向对话智能体的长程记忆研究

长期记忆是提升智能体连续性的重要方向。早期对话系统通常只能依赖有限上下文窗口，因此在多轮交互中容易遗忘用户偏好、历史事件或长期目标。近年来，随着向量检索、记忆库和层级摘要方法的发展，越来越多工作开始为对话系统引入长期记忆模块，以实现跨轮信息保持与长期任务追踪。

现有长期记忆方法大致可分为三类。第一类是基于摘要的记忆，通过对历史对话进行压缩和概括以适应上下文窗口限制；第二类是基于检索的记忆，通过向量数据库或相似度搜索在历史记录中定位相关片段；第三类是层级记忆结构，将短期事件、长期事实和高层反思分层存储，以兼顾细粒度信息与全局连续性。

尽管这些研究为智能体的长期连续性交互提供了重要基础，但在角色对话场景中仍有进一步问题需要解决。角色系统不仅需要记住“事实”，还需要记住“承诺”“关系变化”“冲突修复”等更具叙事意义的信息。本文因此在通用长期记忆框架基础上，提出了更适合角色交互的五层世界线记忆结构，以支撑角色连续性而非单纯事实检索。

## 2.3 可追溯检索与可靠性增强生成

检索增强生成（Retrieval-Augmented Generation, RAG）为对话系统引入了外部知识来源，在提升回答正确性的同时，也带来了来源可信性与归因粒度的问题。若系统只能输出一个“看起来正确”的回答而无法说明结论来源，则其在高可信场景、论文实验和正式展示中的说服力都将受到限制。

近年来，一些研究开始关注检索来源可靠性估计、来源归因和 RAG 评测诊断等问题，试图从“检索到了什么”进一步过渡到“回答中的哪条结论对应哪个来源”。这类工作说明，可靠 RAG 的关键不只是提高命中率，更重要的是建立从回答结论到文档证据的可回查链路。

本文的工作与这类研究相呼应，但更聚焦于角色系统中的应用需求。具体而言，本文并不止于保存 source list，而是通过 `claim_links -> source_refs` 的结构将回答中的具体结论绑定到来源记录，使来源追溯成为后端的正式子系统，并可通过自动评测与消融实验验证其有效性。

## 2.4 智能体记忆安全与提示注入防护

随着 agentic system 的发展，长期记忆安全问题受到越来越多关注。与普通单轮问答不同，智能体一旦将恶意、错误或不受信任的内容写入长期记忆，该污染可能在后续多轮任务中持续放大。提示注入、越权写入、受保护字段覆盖等问题因此成为智能体系统设计中的重要风险点。

针对该问题，已有研究开始探索对记忆写入过程的安全检测、隔离机制与回滚机制。一些方法尝试在工具调用或记忆写入前进行模式匹配与可信度判断，另一些方法则强调将低可信信息写入隔离区，而不是直接进入核心长期记忆层。

本文在此基础上，将 `memory_guard`、`quarantine` 与 `rollback` 一并纳入角色连续体后端。与“模型不要乱记”这类软约束相比，本文更强调在系统层显式阻断危险写入，并为后续调试与实验分析保留可审计链路。

## 2.5 多模态角色交互系统

多模态交互是角色智能体迈向更完整体验的重要方向。语音、视觉形象和摄像头互动等能力可以增强角色的存在感和沉浸感，因此在产品层面具有较高价值。然而，多模态能力也会引入新的系统问题，例如文本与语音状态分叉、实时打断恢复困难以及情绪表达与语义内容不一致等。

现有研究通常分别关注情感语音生成、实时语音交互或多模态情感识别，但对角色连续体系统而言，更重要的是确保多模态能力服从单一后端状态，而不是形成多个相互独立的表达通道。基于这一认识，本文在当前阶段未优先实现完整前端或视觉壳层，而是先完成文本主状态与语音副通道的统一编排，为后续 Live2D 和摄像头交互预留稳定接口。

## 2.6 本章小结

综上所述，现有研究分别在角色对话、长期记忆、检索增强可靠性、记忆安全和多模态交互等方向上提供了重要基础，但对于“角色连续体系统”这一更综合的目标，仍缺乏一个能够同时兼顾人格一致性、世界线连续性、来源追溯和记忆安全的统一后端方案。本文的工作正是在这些研究基础上，围绕二次元 IP 角色场景，对角色智能体系统设计进行了进一步整合与工程化实现。
